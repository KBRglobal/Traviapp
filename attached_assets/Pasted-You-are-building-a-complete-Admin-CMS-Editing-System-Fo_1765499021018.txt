You are building a complete Admin CMS Editing System.
For every page type (Hotels, Attractions, Dining, Districts, Transport, Articles, Homepage, Category Pages, Search Page, 404), the admin must be able to manually edit 100% of the page content.
Do not generate design or content—create only the editable fields, structure, validation rules, and admin workflows.

⸻

GLOBAL EDITING RULES (APPLY TO ALL PAGE TYPES)

SEO FIELDS (required for all pages):
	•	Meta Title (55–60 chars, validator)
	•	Meta Description (150–155 chars, validator)
	•	Canonical URL
	•	Slug editor
	•	OG Title
	•	OG Description
	•	OG Image (upload)
	•	Index/Noindex toggle
	•	Auto SEO score indicator (keyword, length, readability)

	•	Hero Image (upload + alt + title)
	•	H1
	•	Subtitle / Tagline
	•	Primary CTA text
	•	Primary CTA link
	•	Optional badges / labels (rating, category, “from price”)

QUICK INFO BAR:
Each item editable:
	•	Icon selector
	•	Label
	•	Value
	•	Optional tooltip
	•	Optional link

OVERVIEW SECTION:
	•	Short visible intro (3–4 sentences)
	•	Expandable long intro (150–200 words)
	•	WYSIWYG editor
	•	Word count validator
	•	Keyword density validator

FEATURE / HIGHLIGHT CARDS:
Admin can:
	•	Add / delete / reorder cards
Each card fields:
	•	Icon or image upload
	•	ALT text
	•	Title (H3)
	•	Short description (1–2 sentences)

ESSENTIAL INFO GRID:
Admin can edit:
	•	Grid layout
	•	Each card: Icon, Title, Value, Optional small notes

FAQ EDITOR:
	•	Add / delete / reorder questions
	•	Question field
	•	Answer field (100–200 words)
	•	Word count validator
	•	Auto FAQ schema generator

RELATED ITEMS:
	•	Manual selection OR auto-mode
	•	Editable CTA text
	•	Ability to override card data
	•	Reorder items

FINAL CTA SECTION:
	•	CTA headline
	•	CTA button text
	•	CTA link
	•	Benefit bullet list (editable)
	•	Style options (background, layout)

SCHEMA MARKUP BUILDER:
	•	Auto-generate JSON-LD for page type
	•	Editable override for all fields
	•	Types supported: Hotel, TouristAttraction, Restaurant, Article, District, Transport, FAQPage, BreadcrumbList

MEDIA MANAGER:
	•	Upload JPG/PNG/WebP
	•	Auto-optimize (compress + resize)
	•	Editable: ALT, caption, title, filenames
	•	Replace image option

PUBLISHING WORKFLOW:
	•	Save Draft
	•	Preview (desktop + mobile)
	•	Version history + restore
	•	Publish now
	•	Schedule publish
